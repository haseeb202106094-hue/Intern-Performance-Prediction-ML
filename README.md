# 🎯 Intern Performance Prediction Using Machine Learning

## 📌 Overview
Predicting employee/intern performance using machine learning 
based on engagement metrics, training hours, and productivity 
data from 100,000+ employee records.

## 📊 Dataset
- 100,000 employee records
- 20 features including training hours, projects handled,
  satisfaction scores, sick days
- Source: Kaggle — Employee Performance and Productivity Data

## 🛠️ Tools Used
- Python, Pandas, NumPy
- Scikit-learn (Logistic Regression)
- Matplotlib, Seaborn
- Tableau (Dashboard)

## 🔍 Models Built
| Model | Features | Purpose |
|---|---|---|
| Model A | Engagement features only | Task-aligned prediction |
| Model B | Salary + Job features | Full feature prediction |
| Model C | Job/demographic (no salary) | Clean prediction |

## 📈 Key Findings
- Salary is strongly correlated with performance score
- Engagement metrics alone show limited predictive power
- Job title and department influence performance ratings
- Training hours show minimal direct impact on performance score
